# Resume Bullets

## English

- Built an end-to-end customer churn prediction pipeline covering data quality assessment, EDA, leakage-safe preprocessing, model benchmarking, explainability, threshold analysis, and Streamlit/FastAPI deployment.
- Compared Logistic Regression, Random Forest, Gradient Boosting, SVM, and XGBoost; selected tuned XGBoost with ROC-AUC of 0.8467 on a held-out test set.
- Added threshold and lift analysis to translate model probabilities into retention decisions, capturing 78.1% of churners at a 0.30 operating threshold and identifying a top-risk decile with 2.86x lift.
- Packaged the project with reusable Python modules, pytest coverage, CI, generated reports, a model card, data card, and a saved prediction bundle.

## 中文

- 构建端到端客户流失预测项目，覆盖数据质量评估、EDA、防泄漏预处理、模型对比、模型解释、阈值分析以及 Streamlit/FastAPI 部署。
- 对比 Logistic Regression、Random Forest、Gradient Boosting、SVM 和 XGBoost，最终选择 tuned XGBoost，在测试集上达到 0.8467 ROC-AUC。
- 增加阈值与 lift 分析，将概率输出转化为留存运营决策；在 0.30 阈值下捕获 78.1% 流失客户，并识别出 lift 2.86x 的最高风险客群。
- 使用模块化 Python 代码、pytest、CI、自动生成报告、Model Card、Data Card 和模型包提升项目可复现性与工程完整度。
